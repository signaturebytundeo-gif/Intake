8 How to Make Common Changes

Update website content:

  1. Go to GitHub repo
  2. Click on index.html
  3. Click the pencil icon (edit) in the top right
  4. Make your changes
  5. Click "Commit changes" at the bottom
  6. Site will auto-deploy in ~30 seconds

Add new images:

  1. Go to the images/ folder in the GitHub repo
  2. Click "Add file" → "Upload files"
  3. Upload your images and commit
  4. Reference them in index.html as ./images/your-image-name.jpg

Change the Stripe donation link:

  1. Create a new Payment Link at Stripe Dashboard
  2. Edit index.html in GitHub
  3. Search for STRIPE_DONATION_LINK
  4. Replace the URL with your new payment link
  5. Commit the change

Change form notification email:

  1. Log in to Netlify
  2. Go to Site settings → Forms → Form notifications
  3. Edit the email notification hook
